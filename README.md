# AFG ERP · 수수료 정산 대시보드 (Sanitized Demo)

기업 내부 ERP 시스템의 수수료 정산 대시보드 화면을 퍼블리싱했던 작업을 바탕으로,
NDA로 비공개인 원본 화면 대신 **동일한 UI 구조를 가상의 데이터로 재구현한 클린 버전**입니다.

- 원본 스택: Dhtmlx, jQuery, JSP
- 본 데모 스택: HTML / CSS / Vanilla JS + [Chart.js](https://www.chartjs.org/)
- 실적 차트 기간 토글(일간/월간/분기), 계약현황·고객만족도 게이지, 게시판, 퀵메뉴 등 핵심 컴포넌트를 그대로 재현했습니다.
- 화면에 표시되는 모든 수치와 게시글은 샘플 데이터이며 실제 서비스와 무관합니다.

## 실행

별도 빌드 과정 없이 정적 파일로 동작합니다.

```bash
npx serve .
# 또는 index.html을 브라우저로 바로 열어도 됩니다.
```
