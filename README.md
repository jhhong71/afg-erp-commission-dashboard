# AFG ERP · 실적현황 대시보드 (Sanitized Demo)

기업 내부 ERP 시스템의 메인 대시보드("실적현황 / AP순위 / 계약관리 / 관리대상계약 / 게시판") 화면을 퍼블리싱했던 작업을 바탕으로,
NDA로 비공개인 원본 화면 대신 **동일한 UI 구조를 가상의 데이터로 재구현한 클린 버전**입니다.

- 원본 스택: Dhtmlx, jQuery, JSP
- 본 데모 스택: HTML / CSS / jQuery + [Chart.js](https://www.chartjs.org/)
- 전역 상단 메뉴, 실적현황(조직/본인 토글 · 일별 실적 바 차트), AP/팀/지점 순위 테이블, 계약관리(유지율·연체율), 관리대상계약 도넛 차트 4종, 게시판 등 핵심 컴포넌트를 그대로 재현했습니다.
- 우측 상단 배너 영역은 원본 시스템의 실제 홍보 배너 대신 데모용으로 새로 디자인한 그래픽으로 채웠습니다.
- 화면에 표시되는 모든 수치와 게시글은 샘플 데이터이며 실제 서비스와 무관합니다.

## 실행

별도 빌드 과정 없이 정적 파일로 동작합니다.

```bash
npx serve .
# 또는 index.html을 브라우저로 바로 열어도 됩니다.
```
