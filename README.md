# 실적현황 및 관리대상계약 (AFG ERP — Sanitized Demo)

기업 내부 ERP 시스템의 "실적현황 / 관리대상계약" 수수료 화면을 퍼블리싱했던 작업을 바탕으로,
NDA로 비공개인 원본 화면 대신 **동일한 UI 구조를 가상의 데이터로 재구현한 클린 버전**입니다.

- 원본 스택: Dhtmlx, jQuery, JSP
- 본 데모 스택: HTML / CSS / jQuery + [Chart.js](https://www.chartjs.org/)
- 본인/조직 실적 토글, 당월·전월·전년동월·목표 실적 테이블, 일별 실적 바 차트, 관리대상계약(만기리스트·유예/실효예정·당월실효·서류미첨부) 도넛 차트 등 핵심 컴포넌트를 그대로 재현했습니다.
- 내부 프레임워크 호출(`GawebJson`, `Session`, `$.bizServiceCall` 등)은 동작 확인용 Mock 객체로 대체되어 있으며, 화면에 표시되는 모든 수치와 데이터는 샘플입니다.

## 실행

별도 빌드 과정 없이 정적 파일로 동작합니다.

```bash
npx serve .
# 또는 index.html을 브라우저로 바로 열어도 됩니다.
```
